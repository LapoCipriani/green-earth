# GreenEarth

Landing page for the fictional GreenEarth company, developed as part of the start2impact course.  
GreenEarth is a responsive landing page for an environmental startup focused on reforestation, ocean cleanup, and environmental education.


### Technologies employed:

- HTML5
- CSS3
- Flexbox (mobile-first)
- Responsive design
- SVG icons
- Accessibility basics (aria-labels)
